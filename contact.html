Who are we?

This website was completed as our final project for a Code First: Girls course. They work with women to develop their coding skills to help bridge the gender gap in tech.

Visit Code First: Girls for more information and to see when and where their next free community courses are!


I'm Katie and I'm a PhD student in Neuroscience at the University of Sussex.

I usually work on Neurovascular coupling: working out what types of neurons become active when blood vessels dilate.

I've needed to code a little for my PhD, but have always found it really hard! This course has really inspired me and I've learned so much more about coding and how it can be used. Now I'm interested in becoming a web developer!
