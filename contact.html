Who are we?

This website was completed as our final project for a Code First: Girls course. They work with women and non binary people to develop their coding skills to help bridge the gender gap in tech.

Visit Code First: Girls for more information and to see when and where their next free community courses are!


I'm Katie and I'm a PhD student in Neuroscience at the University of Sussex.

I usually work on Neurovascular coupling: working out what types of neurons (cells in the brain) become active when blood vessels dilate.

I've needed to code a little for my PhD, but have always found it really hard to learn by myself. This course has really inspired me and I've learned so much more about coding and how it can be used. Now I'm interested in becoming a web developer!

Find me on Instagram as @brainenergylab, where I talk about my PhD and my experiences learning to code.
